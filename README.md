# Personal Finance & Investment Intelligence Platform

## Overview

A backend finance platform built using Python, FastAPI, SQLite, Docker, and Data Structures & Algorithms concepts.

The platform helps users:

* Track income
* Track expenses
* Manage investments
* Analyze financial health
* Generate budget plans
* Forecast savings
* View dashboard analytics

## Technologies Used

* Python
* FastAPI
* SQLite
* Docker
* Git & GitHub
* Linux (Ubuntu)
* Swagger API Documentation

## Features

### Core Features

* User Management
* Income Tracking
* Expense Tracking
* Investment Tracking

### Analytics Features

* Financial Health Score
* Top Expense Categories
* Budget Planning
* Savings Forecast
* Dashboard Summary
* Investment Summary

* Expense Search
* Expense Sorting
